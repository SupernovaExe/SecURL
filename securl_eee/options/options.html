SecURL: A Machine Learning-based Malicious URL Detector

Security Strength

Will configure the complexity of the detection algorithm to be used in analyzing the active URL. Basic Security analyzes the URL structure with a short detection delay, while Enhanced Security takes into account the response of the link and offers more accuracy at the cost of a higher delay in response time.

Frequently Asked Questions (FAQs)

SecURL is an extension that detects malicious URL. It uses a machine learning model that is trained and optimized using carefully selected URL features and model parameters.
The extension uses either a URL's lexical features (the alphanumeric characters) or its HTML code (the code ran to display a URL's contents), depending on your desired security strength. The model within the extension is trained using hundreds of thousands of URLs to judge a user's input URL's safety based on the features it has.
The extension can adapt to new and emerging threats using concept drift detection. Upon continuous use, there comes a point where the error in the extension's predictions increases, which causes a retraining of the model and reevaluation of the features to occur.
The most common cause of the extension's non-functionality is your connection. Whenever you want the extension to function, ensure that you're either connected to the UPD network or are using OpenVPN.
